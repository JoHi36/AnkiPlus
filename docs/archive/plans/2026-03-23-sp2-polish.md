# SP2 Polish Plan

**Goal:** Fix positioning, layout, and visual issues from the SP2 migration.

---

## Fix 1: Hide TopBar in Review State

**File:** `frontend/src/MainApp.jsx`

The TopBar (Stapel/Session/Statistik tabs) takes 60px and is not useful in review mode. Hide it.

In the render section, wrap TopBar with a condition:
```jsx
{activeView !== 'review' && (
  <TopBar ... />
)}
```

---

## Fix 2: Scope ReviewerView Action Dock to Left Panel

**File:** `frontend/src/components/ReviewerView.jsx`

Currently: `position: fixed; left: 50%; transform: translateX(-50%)` — centers to viewport, overlaps SessionPanel.

Fix: Change from `fixed` to `absolute`, position relative to the ReviewerView container. The parent flex container must have `position: relative`.

```css
/* Action dock: absolute within ReviewerView, centered horizontally */
position: absolute;
bottom: 24px;
left: 50%;
transform: translateX(-50%);
```

Ensure ReviewerView root div has `position: relative`.

## Fix 3: Scope SessionPanel ChatInput to Right Panel

**File:** `frontend/src/components/SessionPanel.jsx`

Currently: `className="fixed bottom-0 left-0 right-0"` — spans full viewport.

Fix: Change from `fixed` to `absolute`, constrained within the SessionPanel container. The SessionPanel wrapper must have `position: relative` and `overflow: hidden`.

```jsx
<div className="absolute bottom-0 left-0 right-0 z-50 px-4 pb-4">
  <ChatInput ... />
</div>
```

## Fix 4: Sidebar Slide-in Animation

**File:** `frontend/src/MainApp.jsx`

Add CSS transition for the sidebar panel appearing when entering review state. Use the same pattern as FreeChat's `freeChatTransition`.

```css
.session-panel {
  transform: translateX(0);
  transition: transform 300ms cubic-bezier(0.25, 0.1, 0.25, 1);
}
.session-panel.hidden {
  transform: translateX(100%);
}
```

---

## Fix 5: Resize Handle for Sidebar

**File:** `frontend/src/MainApp.jsx`

Add a draggable divider between ReviewerView and SessionPanel. On mousedown, track mouse movement and adjust sidebar width. Persist to localStorage.

Simple implementation: 4px wide div between the panels with `cursor: col-resize`. On drag, update CSS custom property `--sidebar-width`.
