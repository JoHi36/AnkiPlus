# Anki AI Tutor - Smart Flashcard Assistant

Ein intelligenter Anki-Addon, der KI-gestützte Lernunterstützung bietet.

## Features

- KI-gestützte Chat-Funktionalität
- Session-Management
- Firebase-Integration
- Modernes React-Frontend
- Cloud Functions für Backend-Services

## Installation

Siehe [SETUP.md](SETUP.md) für detaillierte Installationsanweisungen.

## Entwicklung

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend Functions
```bash
cd functions
npm install
npm run build
```

## Konfiguration

Kopiere `config.json.example` zu `config.json` und fülle die erforderlichen Werte aus.

**Wichtig:** `firebase-service-account.json` und `sessions.json` sind nicht im Repository enthalten und müssen lokal erstellt werden.
